# Qt/QML Snake Game

A classic Snake game implemented using C++20, Qt 6, and QML.

## Features

- Real-time movement using QTimer
- Keyboard controls
- Food generation
- Collision detection
- Dynamic snake growth
- C++ backend with QML frontend
- QAbstractListModel based rendering

## Technologies

- C++20
- Qt 6
- Qt Quick / QML
- CMake
